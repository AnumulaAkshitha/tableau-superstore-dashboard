### 📊 Project Overview
Built an interactive sales performance dashboard using Tableau Public to analyze Superstore retail data. The dashboard visualizes revenue trends, regional performance, and category insights to support data-driven business decisions.

This project demonstrates proficiency in data visualization, dashboard design, and KPI reporting for business analytics roles.

---

### 🎯 Key Business Questions
1. What is the total revenue?
2. Which US region generates the highest sales?
3. Which product category is most profitable?
4. How can users interactively explore data by region?

---

### 📈 Key Insights
1. **Total Revenue:** $2.3M+ in sales across all regions
2. **Top Performing Region:** West region leads in total sales volume
3. **Leading Category:** Technology category generates the highest revenue

---

### 📋 Dashboard Components
1. **KPI Card - Total Revenue:** Currency-formatted metric displaying sum of sales
2. **Sales by Region:** Vertical bar chart comparing Central, East, South, West regions
3. **Sales by Category:** Horizontal bar chart ranking Technology, Furniture, Office Supplies
4. **Interactive Filtering:** Click any region bar to dynamically filter the entire dashboard view
