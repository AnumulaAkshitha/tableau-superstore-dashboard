### Project Goal
Analyze Superstore sales data to identify revenue trends, top-performing regions, and category performance using interactive visualizations.

### Key Insights
1. **Total Revenue:** $2.3M+ across all regions
2. **Top Region:** West region leads in sales volume
3. **Best Category:** Technology generates highest revenue
4. **Order Volume:** 5,000+ total orders processed

### Dashboard Components
1. **KPI Cards:** Total Revenue, Total Orders
2. **Sales by Region:** Bar chart comparing 4 US regions  
3. **Sales by Category:** Horizontal bar chart for 3 product categories
4. **Interactive Filters:** Click any region to filter entire dashboard
